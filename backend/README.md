## Description

Nest JS project starter

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# run init-schema migration
$ npm run publish-migration

# run seeds 
$ npm run seed
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
